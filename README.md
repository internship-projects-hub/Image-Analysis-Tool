# Image-Analysis-Tool




A lightweight image analysis tool for processing image files and computing pixel-level statistics.

## Features

- Load and analyze image files
- Support for PNG and JPG formats
- Calculate image dimensions
- Count total pixels
- Extract pixel-level information
- Compute basic image statistics
- Modular and easy-to-extend architecture

## Project Goals

This project aims to provide a simple foundation for image analysis by implementing core functionality from scratch wherever possible. It is designed as a learning project for understanding image processing fundamentals and pixel-based computations.

## Planned Features

- [ ] PNG image support
